SOLVING THE PUZZLE

It must be your turn to solve

Two methods to solve:

Custom Keyboard

Solve button

Custom Keyboard

Keep entering letters to solve all letters missing

Drawback: Lose points for vowels guessed.

Solve Button

Opens an entry field

Can use keyboard entry, or

Can use voice entry

Advantage: No points lost for vowels.

Winning

The winner is the player with the most points

after all games are played.

Continued play

Solving the last game enables the grey Play Again button.

The activated button is red

This resets the game to start again.
